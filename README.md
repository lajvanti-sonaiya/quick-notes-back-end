# quick-notes-backend

BackAdd is a **quick note-taking application** built with **Node.js**. It allows users to quickly add, view, edit, and delete notes in a simple and efficient way.


## Features

- Add new notes quickly
- View all saved notes with pagination, search and filter
- Edit existing notes
- Delete unwanted notes


## Tech Stack

- **Backend:** Node.js, Express.js ,Typescript 
- **Database:** MongoDB



## Installation

1. Clone the repository:

## HTTP
git clone https://github.com/lajvanti-sonaiya/quick-notes-back-end.git

## SSH

git clone git@github.com:lajvanti-sonaiya/quick-notes-back-end.git

cd quick-notes-back-end

2. Install dependencies
npm install


3. add .env


4. start server

npm start or npm run dev
